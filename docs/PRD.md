# Prompt管理工具 - 产品需求文档

## 1. 项目概述

### 1.1 项目背景
随着AI技术的快速发展，用户在日常工作中需要使用大量的Prompt来与AI进行交互。为了提高工作效率，需要一个集中管理、快速访问和复制的Prompt管理工具。

### 1.2 项目目标
- 提供便捷的Prompt存储和管理功能
- 支持快速搜索和复制Prompt内容
- 通过悬浮球提供全局快速访问能力
- 提供良好的用户体验和现代化的界面设计

### 1.3 目标用户
- AI工具重度使用者
- 需要频繁使用Prompt的开发者、产品经理、内容创作者等
- 希望提高工作效率的团队和个人

## 2. 功能需求

### 2.1 Prompt管理功能

#### 2.1.1 Prompt的增删查改（CRUD）
- **创建Prompt**
  - 支持手动输入Prompt内容
  - 支持从剪贴板粘贴Prompt内容
  - 必填项：Prompt名称、Prompt内容
  - 选填项：备注说明、效果图、标签、分组

- **查看Prompt**
  - 列表视图：侧边栏列表展示所有Prompt
  - 卡片视图：主页面卡片式陈列所有Prompt
  - 详情视图：点击后右侧展示Prompt完整详情

- **编辑Prompt**
  - 支持修改Prompt名称、内容、备注说明
  - 支持更新效果图
  - 支持修改分组和标签

- **删除Prompt**
  - 支持单个删除
  - 支持批量删除
  - 删除前需要确认提示

#### 2.1.2 Prompt属性
- **基本信息**
  - Prompt名称（必填，用于标识和搜索）
  - Prompt内容（必填，实际使用的Prompt文本）
  - 备注说明（选填，用于说明Prompt的用途、使用场景等）
  - 创建时间（自动记录）
  - 更新时间（自动记录）
  - 使用次数（自动统计）

- **分类管理**
  - 分组功能：支持创建、编辑、删除分组
  - 标签功能：支持为Prompt添加多个标签
  - 分组和标签都支持在侧边栏和卡片视图中筛选

- **效果图管理**
  - 支持上传图片（格式：JPG、PNG、GIF）
  - 支持在详情页查看效果图
  - 支持删除效果图
  - 图片大小限制：单张不超过5MB

#### 2.1.3 视图模式
- **侧边栏列表视图**
  - 左侧固定侧边栏，显示所有Prompt
  - 显示Prompt名称
  - 支持按分组筛选
  - 支持搜索功能
  - 点击后右侧显示详情

- **卡片式陈列视图**
  - 主页面网格布局展示Prompt卡片
  - 每个卡片显示：名称、内容预览（前100字符）、标签、分组
  - 支持按分组、标签筛选
  - 支持搜索功能
  - 点击卡片进入详情页

- **详情视图**
  - 右侧或主区域展示Prompt完整信息
  - 显示：名称、完整内容、备注、效果图、分组、标签、创建/更新时间、使用次数
  - 支持编辑和删除操作
  - 支持快速复制按钮

### 2.2 快速复制功能

#### 2.2.1 复制方式
- **一键复制**
  - 在Prompt详情页提供醒目的"复制"按钮
  - 点击后立即复制Prompt内容到剪贴板
  - 复制成功后显示提示信息

- **快捷键复制**
  - 支持快捷键复制（如Ctrl+C）
  - 在列表和卡片视图中支持快捷键操作

- **右键菜单复制**
  - 在列表项和卡片上右键显示菜单
  - 菜单中包含"复制"选项

#### 2.2.2 复制反馈
- 复制成功后显示Toast提示
- 记录复制次数（用于统计使用频率）

### 2.3 悬浮球功能

#### 2.3.1 悬浮球特性
- **位置和样式**
  - 圆形图标，不遮挡主要工作区域
  - 支持拖拽移动位置

#### 2.3.2 功能操作
- **左键点击**
  - 展开/收起Prompt列表面板
  - 面板显示所有保存的Prompt
  - 支持搜索框快速搜索
  - 支持按分组筛选

- **快速复制**
  - 在展开的列表中点击任意Prompt
  - 立即复制该Prompt内容到剪贴板
  - 复制后自动收起面板（可选设置）

- **搜索功能**
  - 面板顶部提供搜索框
  - 实时搜索Prompt名称和内容
  - 支持模糊匹配

#### 2.3.3 设置选项
- 支持设置是否自动收起面板
- 支持设置快捷键打开/关闭悬浮球

### 2.4 搜索功能

#### 2.4.1 搜索范围
- Prompt名称
- Prompt内容
- 备注说明
- 标签

#### 2.4.2 搜索特性
- 实时搜索（输入即搜索）
- 支持模糊匹配
- 高亮显示匹配关键词
- 支持搜索历史记录（可选）

### 2.5 分组和分类功能

#### 2.5.1 分组管理
- 支持创建自定义分组
- 支持编辑分组名称
- 支持删除分组（删除前检查是否有关联的Prompt）
- 支持拖拽Prompt到不同分组
- 支持分组排序

#### 2.5.2 标签管理
- 支持创建标签
- 支持为Prompt添加多个标签
- 支持标签颜色标识
- 支持按标签筛选Prompt

## 3. 非功能需求

### 3.1 性能要求
- 应用启动时间 < 3秒
- Prompt列表加载时间 < 1秒
- 搜索响应时间 < 200ms
- 图片加载支持懒加载

### 3.2 兼容性要求
- 支持Windows 10及以上版本
- 支持macOS 10.14及以上版本（可选）
- 支持Linux主流发行版（可选）

### 3.3 数据安全
- 本地数据加密存储
- 支持数据备份和恢复
- 支持数据导出（JSON格式）

### 3.4 用户体验
- 界面简洁现代，参考orbit-devops风格
- 操作流畅，响应迅速
- 提供操作引导和帮助文档
- 支持暗色/亮色主题切换

## 4. 界面设计要求

### 4.1 设计风格
- 参考网站：https://orbit-devops-40.aura.build/
- 现代化、简洁的设计风格
- 良好的视觉层次和间距
- 统一的色彩方案和图标系统

### 4.2 布局要求
- 响应式布局设计
- 侧边栏宽度可调整
- 主内容区域自适应
- 悬浮球不遮挡重要内容
