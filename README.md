# Prompt管理工具

一个基于Electron的桌面Prompt管理工具，帮助用户高效管理和快速使用Prompt。

## 项目结构

```
prompt-management/
├── PRD.md                  # 产品需求文档
├── TECHNICAL_DESIGN.md     # 技术方案文档
├── README.md               # 项目说明文档
├── backend/                # 后端代码目录
├── frontend/               # 前端代码目录
└── docs/                   # 其他文档目录
```

## 功能特性

- ✅ Prompt的增删查改
- ✅ 分组和标签管理
- ✅ 效果图上传和查看
- ✅ 快速搜索功能
- ✅ 一键复制功能
- ✅ 桌面悬浮球快速访问
- ✅ 侧边栏和卡片式视图

## 技术栈

- **前端**: Electron + React/Vue
- **后端**: Python (FastAPI)
- **数据库**: PostgreSQL
